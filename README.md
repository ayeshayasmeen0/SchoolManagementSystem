# School Management System

A simple Student Management System using C# WinForms and SQLite.

## Features

- Add Student
- Update Student  
- Delete Student
- Search Student
- View All Students

## How to Run

1. Open project in Visual Studio
2. Install NuGet package: `Install-Package System.Data.SQLite.Core`
3. Press F5 to run

## Login (No login required)

Direct form opens. Sample data auto-creates.
